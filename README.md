# Rippler

Десктопное приложение для шифрования текста с использованием модулярной арифметики (аффинное преобразование) и сжатия файлов методом Хаффмана.  
Интерфейс на русском языке, построен на **PyQt**, все функции доступны на главном экране.

## Возможности

- **Affine**:
    - Шифрование текста с привязкой к короткому 4‑значному коду `DDHH` (день+час).
    - Дешифрование с проверкой целостности по SHA-256.
    - Поддержка многострочного ввода (несколько слов разделяются пробелами) — каждое слово шифруется отдельно и выводится в отдельной строке.
    - Копирование шифра, кода и модуля в буфер обмена.
- **Huffman**:
    - Кодирование (сжатие) произвольных файлов в формат `.huff`.
    - Декодирование (восстановление) файлов из `.huff`.
- **FAQ** — подробное описание работы программы (открывается по кнопке внизу окна).

## Алгоритм Affine (шифрование)

1. **Алфавит** — фиксированная строка из 128 символов:

```
АБВГДЕЁЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ
абвгдеёжзийклмнопрстуфхцчшщъыьэюя
ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz
0123456789
```

Индексы символов — от 0 до 127.

2. **Генерация ключа** из кода `DDHH` (например, `1423`):
    - Код преобразуется в байты UTF-8.
    - Вычисляется SHA-256 от этих байт, получается 32 байта ключа `key_bytes`.

3. **Шифрование одного символа** (с индексом `x`, позиция в тексте `i`):
   y = (x + key_bytes^i mod 32)mod M, где `M` — заданный пользователем модуль.

4. **Формат вывода**:
   Для каждого слова формируется отдельная строка:
   M:числа_через_пробел|SHA256(слово+код)

`

5. Дешифрование:
    - Из каждой строки извлекаются M, список чисел и ожидаемый хеш.
    - По коду восстанавливаются key_bytes.
    - Для каждого числа y:
      x = (y - key_bytes^i mod 32) mod M
    - Если x в пределах 0..127, получаем символ; иначе ошибка.
    - Проверка хеша: SHA-256(восстановленное_слово + код) должен совпасть с хешем из строки.

## Алгоритм Huffman

Реализовано классическое кодирование Хаффмана для файлов:

- Построение дерева частот.
- Запись сжатых данных вместе с таблицей для восстановления.
- Декодирование с точным восстановлением исходного файла.

## Требования и запуск

- Python 3.7 или выше.
- Установите PyQt5:

```
pip install PyQt5
```

- Для запуска из исходного кода:

```
python main.py
```

- Для сборки .exe под Windows:
    1. Установите PyInstaller: pip install pyinstaller
    2. Запустите build_windows.bat или выполните:
    ```
     pyinstaller --onefile --windowed main.py
    ```
    3. Исполняемый файл будет в папке dist.

## Пример

Шифрование текста "привет мир" с модулем 200 (код 1215):

Входной текст:

```
привет мир
```

Результат (две строки шифра):

```
200:45 123 78 89 12 200|a1b2c3d4e5f6...
200:34 56|f6e5d4c3b2a1...
```

Дешифрование тех же строк с кодом 1215 вернёт:

```
привет
мир
```
