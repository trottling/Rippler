# Rippler

## Назначение приложения
Rippler — это консольное приложение на Python для шифрования и дешифрования текста в ограниченном алфавите с использованием собственного текстового формата сообщений.

## Запуск
1. Установите Python 3.8+.
2. Перейдите в папку проекта.
3. Запустите приложение:

```bash
python main.py
```

## Ограничения алфавита
Приложение работает только с символами, входящими в поддерживаемый алфавит (буквы, цифры и разрешённые служебные знаки). Если во входной строке встречаются неподдерживаемые символы, их нужно удалить или заменить до шифрования.

## Формат сообщений `M:...|...`
Шифротекст хранится в формате:

```text
M:<payload>|<checksum>
```

Где:
- `M:` — префикс типа сообщения;
- `<payload>` — зашифрованные данные;
- `|` — разделитель;
- `<checksum>` — контрольная часть для проверки целостности.

## Пример шифрования/дешифрования
Пример пользовательского сценария:

- Исходный текст: `HELLO123`
- После шифрования: `M:KHOOR456|9A`
- После дешифрования: `HELLO123`

> Пример иллюстративный: фактический результат зависит от алгоритма, реализованного в `main.py`.

## Сборка для Windows 7+
Ниже шаги для сборки `main.py` в `.exe` через PyInstaller.

1. Установите Python для Windows (рекомендуется 3.8–3.11).
2. Откройте `cmd` в корне проекта.
3. Создайте и активируйте виртуальное окружение:

```bat
python -m venv .venv
.venv\Scripts\activate
```

4. Установите зависимости:

```bat
pip install --upgrade pip
pip install pyinstaller
```

5. Запустите сборку:

```bat
build_windows.bat
```

6. Готовый `.exe` появится в папке:

```text
dist\Rippler.exe
```

Альтернативно можно собрать напрямую:

```bat
pyinstaller --clean --noconfirm pyinstaller.spec
```
