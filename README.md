# Rippler

Десктопное приложение на Python/Tkinter для шифрования и дешифрования текста по модульной арифметике с 4-символьным кодом `DDHH` и проверкой целостности SHA-256.

## Возможности

- Разделы: **Шифрование**, **Дешифрование**, **FAQ**.
- Бургер-меню `☰` для переключения разделов.
- Интерфейс на русском языке.
- Копирование/вставка (Ctrl+C/V/X и контекстное меню).
- Кнопки копирования шифра, кода и результата.

## Алгоритм

- Алфавит фиксирован из 128 символов (русский + английский + цифры).
- Код `DDHH` (день+час) хешируется `SHA-256`, получаются 32 байта ключа.
- Для каждого символа с индексом `idx`:
  - `c = (idx + key_bytes[i % 32]) % M`.
- Формат шифра:
  - `M:числа_через_пробел|hex_sha256(текст+код)`.
- При дешифровании:
  - `idx = (c - key_bytes[i % 32]) % M`;
  - если `idx >= 128` — ошибка;
  - затем сверяется SHA-256 хеш, иначе ошибка: `Неверный код или повреждённые данные`.

## Запуск

```bash
python main.py
```

## Сборка `.exe` (Windows 7+)

1. Установить PyInstaller:

```bash
pip install pyinstaller
```

2. Выполнить сборку:

```bash
build_windows.bat
```

Результат будет в папке `dist`.
